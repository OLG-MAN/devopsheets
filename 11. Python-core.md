#### Operators Priority
Highest to --> lowest; operators from one group execute from left hand.
```
unary -->  **  -->  * / // %  -->  + -
```

#### Example
```

print((10 - 6) ** 2 / 8 * 10 + 1) -> 21;
```

### Bitwise Operators
```
# Bitwise Operators
&  # Conjunction
|  # Disjunction 
~  # Negation
^  # Exclusive

& # If two bits 1 -> 1; (AND)
| # If one of 2 bits 1 -> 1; (OR)
^ # If one of 2 bits 1 -> 1; If 2 bits 1 -> 0; (XOR)
~ # Change bits;
```

```
# Example with &
15 & 22

print(bin(15))
0b1111
print(bin(22))
0b10110

15 0 0 0 0 1 1 1 1
22 0 0 0 1 0 1 1 0
->
6  0 0 0 0 0 1 1 0

15 & 22
# 6
```
